# Hướng Dẫn Triển Khai Bot Tóm Tắt Telegram

Bot của bạn đã được lập trình để tóm tắt 500 tin nhắn gần nhất khi bạn dùng lệnh `/summary`, lưu trữ tin nhắn vào **Supabase** để phân tích, và chạy **Cron Job** qua máy chủ Vercel vào lúc 9h sáng giờ Việt Nam (2h00 UTC) để đọc và xoá tin nhắn báo cáo hàng ngày. Nó đang sử dụng thuật toán miễn phí **Gemini 1.5 Flash** của Google để tạo tóm tắt cực kì nhanh chóng và chất lượng cao bằng tiếng Việt.

## 1. Thiết Lập Supabase (Database)
1. Đăng ký/Đăng nhập tại [Supabase](https://supabase.com/).
2. Tạo Project mới.
3. Vào phần **SQL Editor** trong Supabase, chạy đoạn lệnh sau:
```sql
CREATE TABLE messages (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  chat_id BIGINT NOT NULL,
  user_id BIGINT NOT NULL,
  user_name TEXT,
  text TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);
```
4. Vào Project Settings -> **API**, sao chép **Project URL** và **anon public key**.

## 2. Thiết Lập Telegram Bot
1. Tìm tài khoản `@BotFather` trên Telegram.
2. Gõ `/newbot` và tạo lấy API Token.
3. Mở cài đặt bot trong BotFather: vào mục **Bot Settings** => **Group Privacy** => Chọn **Turn Off**. Điều này rất QUAN TRỌNG, nó cho phép bot đọc được tất cả tin nhắn chat của mọi người trong group để lưu vào database, thay vì chỉ đọc tin nhắc đến (tag) nó.
4. Thêm bot vào Group của bạn.

## 3. Lấy Gemini API Key
1. Vào [Google AI Studio](https://aistudio.google.com/).
2. Tạo API Key miễn phí mới cho project của bạn.

## 4. Đẩy project lên Vercel
1. Đăng nhập [Vercel](https://vercel.com) và tạo một Project mới.
2. Thiết lập các **Environment Variables** sau theo file `.env.example`:
   - `TELEGRAM_BOT_TOKEN`: Token lấy từ BotFather
   - `SUPABASE_URL`: Project URL từ Supabase
   - `SUPABASE_KEY`: anon public key từ Supabase
   - `GEMINI_API_KEY`: Key từ Google AI Studio
   - `CRON_SECRET`: Bạn tự tạo một chuỗi mật khẩu bí mật bất kỳ ví dụ: `my-super-secret-cron-key-1234`.
3. Bấm **Deploy**. Vercel sẽ thiết lập tự động file `vercel.json` chạy code Python.

## 5. Cấu hình Webhook
Sau khi Deploy xong, Vercel sẽ cho bạn 1 thẻ domain (ví dụ: `https://ten-du-an.vercel.app`). Mở trình duyệt và truy cập URL mẫu này để nhúng Webhook vào Telegram:
```
https://api.telegram.org/bot<TELEGRAM_BOT_TOKEN_CỦA_BẠN>/setWebhook?url=https://ten-du-an.vercel.app/api/webhook
```
> Nếu trả về `{ "ok": true, "result": true, "description": "Webhook was set" }` là thành công!

## 6. Lời khuyên cho Cronjob
Bởi vì Vercel Cron sẽ gọi Endpoint `/api/cron` bằng Method GET dựa trên lịch đặt tại `vercel.json` (0 2 * * * = 9h VN). Nó cũng sẽ chèn CRON_SECRET như bạn đã bảo mật để tránh người khác kích hoạt. Bạn không cần thiết lập thêm gì, mọi thứ sẽ chạy tự động một cách mượt mà. Đảm bảo bạn đã cấp CRON_SECRET trong mục Environment Variables trùng khớp!
